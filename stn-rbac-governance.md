# STN Dashboard RBAC Governance Policy

이 문서는 STN 대시보드의 권한 기반 접근 제어(RBAC) 원칙을 정의하며, 모든 UI/UX 수정 시 이 규칙을 최우선으로 준수해야 합니다.

## 1. 역할 정의 (Roles)
- **admin**: 시스템 관리자. 모든 편집, 설정, 프로젝트 관리 권한 보유.
- **user**: 일반 사용자. 대시보드 조회 전용 권한 보유 (Read-Only).

## 2. GNB (Global Navigation Bar) 가시성 규칙
관리자(`admin`)가 아닌 사용자에게는 다음 요소들을 절대로 노출하지 않는다:
- 차트 라이브러리 선택 드롭다운 (amCharts 등)
- **Design** 버튼 및 사이드바
- **Layout** 버튼 및 사이드바
- **Edit** 모드 전환 버튼
- **Preview** 모드 전환 버튼 (일반 사용자는 Preview 모드가 고정 상태임)

## 3. 기본 화면 모드 (Default View Mode)
- 일반 사용자(`user`) 로그인 시, 시스템은 즉시 `isPreviewMode = true`를 강제 적용한다.
- 일반 사용자는 Preview 모드를 해제할 수 있는 UI 수단을 제공받지 않는다.

## 4. 위젯 상호작용 규칙
일반 사용자(`user`) 화면에서 다음 기능을 차단한다:
- 위젯 우측 상단의 편집/삭제 버튼 숨김
- 위젯 드래그 앤 드롭 (이동) 차단
- 위젯 리사이징 (크기 조절) 차단

## 5. 변경 관리
이 권한 체계를 수정하거나 우회하는 코드를 작성할 경우, 반드시 사전에 관리자의 명시적인 승인을 얻어야 한다.
