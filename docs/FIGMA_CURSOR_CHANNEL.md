# Cursor ↔ Figma Plugin 연동

Cursor와 Figma 플러그인 간 통신에 사용하는 채널 ID입니다.

## 채널 ID

```
4y002t6r
```

플러그인 또는 Cursor 측에서 "Figma plugin channel" / "Cursor talk to Figma" 설정 시 위 ID를 입력하면 됩니다.

- 디자인 토큰을 Figma 변수로 내보내려면 앱 내 **Design** 패널 → **Figma 연동** → "Figma 변수로 내보내기"를 사용하세요.
